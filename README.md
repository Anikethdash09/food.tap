# Food App

Welcome to the Food App project! This Java Full Stack application allows you to create account and to order food online.

# Features
- User Registration & Login
- Account Creation
- Multiple restaurant
- New restaurant can be added
- Order food
- Rate of food, Cost and timing 
- Pay on any mode
- Get delivery time
- Give us feed back

# Technologies Used
- Java
- JDBC 
- JSP
- Servlets
- Spring
- Hibernate
- MySQL
- HTML
- CSS
- JavaScript
